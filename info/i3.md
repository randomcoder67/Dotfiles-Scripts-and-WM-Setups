# i3 Window Manager Config

![Screenshot of my setup](i3.png)

## Installation

In addition to the regular installation commands, run:  
`./init.sh --install i3`  
`./setup.sh --make i3`  

## Features

### Panel

Notable features

* System monitors:
	* Network Status
	* Battery
	* CPU Usage
	* CPU Temperature
	* RAM Usage
* Time Info:
	* Current Date
	* Time in Multiple Timezones (Shown: New York - London - Seoul)
* Livestream Monitors
* Volume
